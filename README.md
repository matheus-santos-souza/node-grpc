Node.js com TypeScript e gRPC
================================

Este projeto é uma aplicação Node.js utilizando TypeScript e gRPC.

Clone o repositório
------------------------
```bash
git clone https://github.com/matheus-santos-souza/node-grpc.git
```

Rodando o Projeto
------------------------
```bash
docker compose up -d
```

Tecnologias Utilizadas
----------------------

*   Node.js v20.10.0
*   npm v10.2.3
*   TypeScript v5.3.3
*   TSX
*   Docker
*   grpc-js: 
*   grpc-proto-loader
*   grpc-reflection
